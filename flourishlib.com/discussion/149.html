This would be an issue with your PHP setup, you may want to check what you have set for 'display_errors' in your php.ini (it should be set to 1).

Or, try placing this "ini_set('display_errors', 1);" right before the "include './inc/init.php';" on the demo site's index.php.

If that displays errors, you are going to want to change your php settings or place that line of code in a config file, so you don't have to repetitively redefine it.
