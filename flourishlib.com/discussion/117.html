i want to build a search like you have in this site. have you some advises for me. Maybe a little guideline pls.

The search on this site is the standard Trac search with some relevance ranking improvements.

  1. Find all pages that contains the words in question
  2. Rank the result. Pages are ranked highest if a search word is in the URL, second highest if the word is in the H1, and third highest by the number of occurrences in the body content. If there are multiple matches per category, ties are broken by the number of occurences.
