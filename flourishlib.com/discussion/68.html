Can you do a few things?

First, expose the fDatabase instance so I can see what database extension is being run: 

fCore::expose(fORMDatabase::retrieve());

Can you turn on database debugging and paste all of the database queries being executed? 

fORMDatabase::retrieve()->enableDebugging();

Also, right before you call store(), can you expose the record? 

fCore::expose($record);
$record->store();

I'm thinking this might be related somehow to ticket #201.
