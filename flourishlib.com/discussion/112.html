fRequest::get('nume', 'array', array(1)); how can exactly use this function to create an array from an input field, i want to use this for a search with more than 1 words.

This is an @internal public method, which means it may be changed in the future, however it will split a string into an array of search terms: 

$words = fORMDatabase::parseSearchTerms('This is the string!');
// $words will be aray('This', 'is', 'the', 'string');

This function is automatically applied to a string that is passed to the ~ operator in an fRecordSet::build() call. You can see fRecordSet#Operators for more info.
