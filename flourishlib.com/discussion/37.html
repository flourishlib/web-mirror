DOCUMENT_ROOT

posted by brewho 10 years ago

I was having trouble with DOCUMENT_ROOT because my test environment (Aptana on Windows XP) doesn't define this. I came up with this, which seems to be working at the moment. Is there perhaps a better solution? 

if(!isset($_SERVER['DOCUMENT_ROOT'])){
    $me = str_replace('%20',' ',dirname($_SERVER['PHP_SELF']));
    $path = str_replace('\\\\','/',__FILE__);
    $_SERVER['DOCUMENT_ROOT'] = substr($path,0,strpos($path,$me));
}

I'm not sure how Aptana executes code, however when I've run into such issues before (running PHP in CGI mode on Windows), I usually hard code the document root into my init/bootstrap file such as below: 

$_SERVER['DOCUMENT_ROOT'] = '/path/to/document/root'
