posted by wbond 9 years ago

I can't speak for the other frameworks, but obviously I can speak for inKWell since I'm the one working on it. It is in a very usable state, although I'm battling with finding the time to write better documentation and to polish the website for an actual beta release. I'm always available to answer questions directly though, and the more extensive and updated documentation thus far is in the github wiki, not on the site which is not extensively developed.

One additional note, the inkwell_site branch on github has a bit more to offer than the master branch, as it has what eventually will be forked out as official extensions, including the stock CMS and basic PagesController
