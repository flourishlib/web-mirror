This Costuri can not be deleted because:

  • One or more Facturi Agenties references i

i receive this even facturi agenties is empty. how can i force deletion?

If the ORM thinks you still have some related records, it probably means you deleted the records without using the ORM.

To force delete, even when there is an ON DELETE RESTRICT clause, pass TRUE to the ->delete() method of fActiveRecord. 

// Delete the record and all records that reference it (recursively)
$record->delete(TRUE);
