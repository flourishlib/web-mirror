posted by wbond 10 years ago

@Chris

I thought I'd just let you know that Oracle support was completed a few weeks ago. It is now possible to use fDatabase, fSQLTranslation, fSchema and the ORM with Oracle databases.

Everything has been tested with 10g XE, however from reading information online it seems that pretty much all of the features should be supported by Oracle 8 and 9i too.

I've implemented support for the OCI8, PDO_OCI, ODBC and PDO_ODBC drivers. It seems that unixODBC on Linux doesn't like some of the ALTER SESSION queries I run when setting up the connection, but the ODBC connections work fine on Windows.
