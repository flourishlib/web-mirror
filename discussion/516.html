Customize error message

posted by jmtucu 8 years ago

Hi! I am currently using the following code to capture errors: 

fCore::enableErrorHandling(STORAGE_PATH.'logs'.DS.'errors.log');
fCore::enableExceptionHandling(STORAGE_PATH.'logs'.DS.'exceptions.log');
fCore::disableContext();

That works perfect, if an error occurs they are stored in those files. In the view the system shows the message 

It appears an error has occurred - we apologize for the inconvenience. The problem may be resolved if you try again.

My question is, how can I customize that message or redirect to a custom view, for example /notification/error. I do not want to change the flourish files because if I upgrade I lose those changes...

Thanks!

fCore::enableExceptionHandling() accepts three parameters, two more than you have provided. The second parameter is a callback to execute after catching the exception. The third is an array containing the values to pass to the callback as parameters.

If you set the callback, the default "handler" that prints the text will not get executed.
