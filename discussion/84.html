posted by richard 10 years ago

i have a post classes that right now returns all the posts i have made the titles into links but using the flourish way i can't for the life of me figure out how to handle that so the user can view that individual record. any help?

Lets assume we start with the following code: 

fORMDatabase::attach(new fDatabase());

class Post extends fActiveRecord {
    static public function buildAll() {
        return fRecordSet::build(__CLASS__);
    }
}

To create a new post: 

$post = new Post();

To load a post out of the database with a post_id (or whatever your primary key is) of 3: 

$post = new Post(3);

To create all posts: 

$posts = Post::buildAll();
