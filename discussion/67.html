Thumb

posted by dragos 10 years ago

can u do some improvement like when a resize an image to have posibilities to save that thumbnails in other directory, to preserve some thumbs and still have the big images?

This is currently possible manually with the fImage class or via the fORMFile class when using the ORM.

With fImage, simply do: 

$upload = new fUpload();
$image = $upload->move('image', '/path/to/main/photo/');
$thumb = $image->duplicate('/path/to/thumb/photo/);
$thumb->resize(100, 100);
$thumb->saveChanges();

With the ORM you would want to use various methods of the fORMFile class. The fORMFile class documentation page has examples you can follow.
