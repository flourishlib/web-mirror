Is there any way to achieve clean urls without specifically using rewrite rules on them all via htaccess as done in the demo site?

Thanks!

Currently there is no functionality built into Flourish to handling routing of requests. You can either use mod_rewrite, like I did with the demo site, or do some simple routing in PHP.

Either one is probably gonna use regular expressions in some form or another. With PHP you could abstract the regex behind some nicer syntax as a tiny little domain specific language (DSL). If I get a chance to later tonight I'll try to post an example.
