Storing error/exceptions messages as XML/RSS

posted by o 10 years ago

I want to follow error / exceptions messages from private rss looks like this format 

<item>
<title>fSQL Exception</title>
<link>http://website.com</link>
<pubDate>Mon, 07 Dec 2009 00:51:40 +0200</pubDate>
<description>// error message</description>
</item>

I think you will probably have to use set_error_handler() and set_exception_handler() and write some callback that write to an XML file. That is what fCore does for its error/exception handlers.
