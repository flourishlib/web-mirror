The !SetCreator class was simply an example in some of the documentation and is not a real class. If you are looking to create record sets, please see the fRecordSet class.

You can definitely use fRecordSet::build() to create a sorted record set. The third parameter accepts the ordering statements.

There currently is no code for pagination, mostly because it is very specific to the HTML of your site.

Finally, there is the Flourish Demo Site if you want to see a real site built with Flourish.
