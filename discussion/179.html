what option do i have to make populate method os store to store my data in base64_encode?

You would need to override the set{ColumnName}() and get{ColumnName}() methods to base64_encode and base64_decode: 

class User extends fActiveRecord
{
    public function getPhoto()
    {
        return base64_decode($this->get('photo'));
    }

    public function setPhoto($photo_file_contents)
    {
        return $this->set('photo', base64_encode($photo_file_contents));
    }
}

fActiveRecord::populate() will call the setPhoto() method and pass the data from the request.
