We use UTF-8 all the time here at the company I work for. Here are the main steps to ensure everything is UTF-8:

  1. For MySQL, set the encoding to UTF-8 for each CREATE TABLE statement. For PostgreSQL, do it when creating the database.
  2. Make sure you specify that the Content-type HTTP header is specifying that the encoding is UTF-8. fHTML::sendHeader() does this.
  3. Add the charset meta tag to the HTML, which you have
