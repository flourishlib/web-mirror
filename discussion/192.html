Using flourish with my own code

posted by tomasberglund 10 years ago

Hi, I'm about to start a medium sized project and I've looked at several frameworks to help me out. But I want to implement my own "old" classes and really don't want MVC and stuff, so Flourish seems great so far. To my question, if I want to use Flourish for say Authentication is it possible to seemlessly use Flourish with my own code? (without hacks so to speak)

Regards Tomas Berglund

You should be able to. Flourish is fairly modular, and the AdvancedDownload page has a way to download only the classes you need. The only caveat is that Flourish expects all text to be in UTF-8, even stuff such as the database contents. The UTF-8 page has some more info.
